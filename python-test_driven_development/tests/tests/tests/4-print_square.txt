=============================
How to use 4-print_square.py
=============================

This module defines ``print_square(size)`` that prints a square using ``#``.

Setup
=====

::

    >>> print_square = __import__('4-print_square').print_square

Normal usage
============

Square of size 4:

::

    >>> print_square(4)
    ####
    ####
    ####
    ####

Square of size 1:

::

    >>> print_square(1)
    #

Square of size 0 (prints nothing):

::

    >>> print_square(0)

Square of size 10:

::

    >>> print_square(10)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
