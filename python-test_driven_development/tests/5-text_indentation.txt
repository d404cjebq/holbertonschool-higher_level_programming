================================
How to use 5-text_indentation.py
================================

This module defines ``text_indentation(text)`` that prints text with 2 new
lines after each ``.``, ``?``, or ``:``.

Setup
=====

::

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Normal usage
============

Text with a period:

::

    >>> text_indentation("Hello.")
    Hello.
    <BLANKLINE>

Text with a question mark:

::

    >>> text_indentation("How are you?")
    How are you?
    <BLANKLINE>

Text with a colon:

::

    >>> text_indentation("This is a list:")
    This is a list:
    <BLANKLINE>

Multiple delimiters:

::

    >>> text_indentation("Hello. How are you? I am fine: thank you.")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    I am fine:
    <BLANKLINE>
    thank you.
    <BLANKLINE>

No leading spaces on new lines (spaces after delimiter are stripped):

::

    >>> text_indentation("Hello.   Next line.")
    Hello.
    <BLANKLINE>
    Next line.
    <BLANKLINE>

Text with no delimiters (prints as-is with no trailing newline):

::

    >>> text_indentation("No delimiters here")
    No delimiters here

Empty string:

::

    >>> text_indentation("")

Text ending without a delimiter:

::

    >>> text_indentation("Hello. World")
    Hello.
    <BLANKLINE>
    World

Invalid argument
================

text is not a string:

::

    >>> text_indentation(42)
    Traceback (most recent call last):
    TypeError: text must be a string

    >>> text_indentation(None)
    Traceback (most recent call last):
    TypeError: text must be a string

    >>> text_indentation(["Hello."])
    Traceback (most recent call last):
    TypeError: text must be a string

================

::

    >>> text_indentation()
