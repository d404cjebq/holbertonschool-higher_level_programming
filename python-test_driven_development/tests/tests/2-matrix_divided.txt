================================
How to use 2-matrix_divided.py
================================

This module defines ``matrix_divided(matrix, div)`` that divides all elements
of a matrix by ``div``, rounded to 2 decimal places.

Setup
=====

::

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Normal usage
============

Dividing a matrix of integers:

::

    >>> matrix_divided([[1, 2, 3], [4, 5, 6]], 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Dividing a matrix of floats:

::

    >>> matrix_divided([[1.5, 2.5], [3.5, 4.5]], 2)
    [[0.75, 1.25], [1.75, 2.25]]

Dividing by a float divisor:

::

    >>> matrix_divided([[1, 2], [3, 4]], 1.5)
    [[0.67, 1.33], [2.0, 2.67]]

Original matrix is not modified:

::

    >>> m = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(m, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
    >>> m
    [[1, 2, 3], [4, 5, 6]]
