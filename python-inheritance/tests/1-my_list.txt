# The `MyList` class

## Using MyList

> > > MyList = **import**('1-my_list').MyList
> > > my_list = MyList()
> > > my_list.append(1)
> > > my_list.append(4)
> > > my_list.append(2)
> > > my_list.append(3)
> > > my_list.append(5)
> > > print(my_list)
> > > [1, 4, 2, 3, 5]
> > > my_list.print_sorted()
> > > [1, 2, 3, 4, 5]
> > > print(my_list)
> > > [1, 4, 2, 3, 5]

## Empty list

> > > my_list = MyList()
> > > print(my_list)
> > > []
> > > my_list.print_sorted()
> > > []

## Negative numbers

> > > my_list = MyList()
> > > my_list.append(-1)
> > > my_list.append(4)
> > > my_list.append(-3)
> > > my_list.append(2)
> > > print(my_list)
> > > [-1, 4, -3, 2]
> > > my_list.print_sorted()
> > > [-3, -1, 2, 4]
> > > print(my_list)
> > > [-1, 4, -3, 2]
