==================
How to use MyList
==================

First, import the class:
    >>> MyList = __import__('1-my_list').MyList

Instantiate the class and add some integers:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

Verify the original list order:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test the `print_sorted` method:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Verify that the original list was not modified by the sort:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Test with negative numbers:
    >>> neg_list = MyList([-1, -5, 0, 2])
    >>> neg_list.print_sorted()
    [-5, -1, 0, 2]
