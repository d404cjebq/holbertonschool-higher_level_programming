===========================
How to use 0-add_integer.py
===========================

This module defines a function ``add_integer(a, b=98)`` that adds two integers.

Usage
=====

``add_integer`` returns the sum of its two arguments as an integer.

::

    >>> add_integer = __import__('0-add_integer').add_integer

Normal integer addition:

::

    >>> add_integer(1, 2)
    3

    >>> add_integer(100, -2)
    98

Default value of b is 98:

::

    >>> add_integer(2)
    100

Floats are cast to integers before addition:

::

    >>> add_integer(100.3, -2)
    98

    >>> add_integer(1.9, 2.9)
    3

Non-integer/non-float raises TypeError:

::

    >>> add_integer(4, "School")
    Traceback (most recent call last):
    TypeError: b must be an integer

    >>> add_integer(None)
    Traceback (most recent call last):
    TypeError: a must be an integer

    >>> add_integer("hello", 2)
    Traceback (most recent call last):
    TypeError: a must be an integer

Float overflow raises OverflowError:

::

    >>> add_integer(float('inf'), 1)
    Traceback (most recent call last):
    OverflowError: cannot convert float infinity to integer

Float NaN raises ValueError:

::

    >>> add_integer(float('nan'), 1)
    Traceback (most recent call last):
    ValueError: cannot convert float NaN to integer
